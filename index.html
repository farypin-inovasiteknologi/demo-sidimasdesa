PETUNJUK TEMPLATE WORD:
  • Gunakan font standar (Arial / Times New Roman / Calibri). DILARANG pakai Text Box.
  • Ekstensi file wajib .docx. Unduh Contoh Di Sini
  • Satu file doxc hanya untuk satu nomor surat
  • Ketik kode wajib ini di dalam file Word Anda agar terisi otomatis:
    {NOMOR_SURAT} : Untuk letak Nomor Surat
    {TEMPAT_TTD} : Untuk Tempat ttd Surat
    {TANGGAL_TTD} : Untuk Tanggal Surat
    {QR_TTE} : letakkan di atas nama Kades/Sekdes (Sistem otomatis mengatur jaraknya, jangan beri terlalu banyak Enter!)
    {TTD_NAMA} : Untuk Nama Kepala Desa
    {TTD_PANGKAT} : Untuk Pangkat/Golongan
    {TTD_NIP} : Untuk NIP Kepala Desa (bila ada)

Kebijakan Privasi & Ketentuan Layanan

SiDiMAS (Sistem Digital Manajemen Arsip Surat)
Versi TTE Mandiri & AI

1. Pendahuluan

Kami sangat menghargai privasi dan keamanan data Anda. Dokumen ini menjelaskan pengelolaan data serta batasan tanggung jawab hukum (Disclaimer) atas penggunaan aplikasi SiDiMAS.

2. Pengumpulan & Penyimpanan Data

SiDiMAS beroperasi di lingkungan ekosistem Google Workspace milik Instansi Anda sendiri. Seluruh data (Database Surat, File PDF, dan Log Aktivitas) tersimpan secara eksklusif dan aman di Google Sheets dan Google Drive Anda. Kami (Pengembang) tidak menyalin, mengambil, atau menyimpan data Anda di server pihak ketiga mana pun.

3. Penggunaan Kecerdasan Buatan (AI)

Fitur Auto-Fill AI menggunakan layanan Google API secara aman. Dokumen yang Anda unggah hanya diproses untuk ekstraksi teks secara real-time dan tidak digunakan untuk melatih model AI publik atau disimpan oleh pengembang.

4. Status TTE & Disclaimer Hukum (UU ITE)

  • Aplikasi ini menghasilkan Tanda Tangan Elektronik (TTE) Tidak Tersertifikasi yang dilengkapi sistem Visual Integrity Check (QR Code & Hash) untuk validasi mandiri.
  • Penggunaan TTE ini direkomendasikan untuk administrasi internal dan operasional persuratan rutin. Untuk dokumen berisiko hukum tinggi atau sangat rahasia, disarankan menggunakan Tanda Tangan dan Stempel Basah.
  • Segala bentuk penyalahgunaan kewenangan, pemalsuan isi dokumen (manipulasi PDF), atau sengketa hukum yang timbul dari penggunaan surat yang dihasilkan oleh aplikasi ini adalah sepenuhnya tanggung jawab Pengguna/Instansi. PT Farypin Inovasi Teknologi dibebaskan dari segala tuntutan hukum.

5. Keamanan Akun

Admin instansi bertanggung jawab penuh untuk menjaga kerahasiaan Username dan Password pengguna. Sistem kami menggunakan enkripsi SHA-256 untuk melindungi kata sandi di dalam database.
